# Executive Sales Performance & Predictive Analytics Dashboard

An end-to-end Business Intelligence web application built with **Python**, **Streamlit**, and **Plotly** to track retail sales performance, analyze commodity metrics, and forecast revenue using statistical modeling.

## 🚀 Live Demo
You can test the interactive live dashboard directly here: [PASTING_LINK_STREAMLIT_KAMU_DI_SINI]

## 🧠 Features & Analytics Capabilities
* **Financial Trends & Performance:** Descriptive analytics covering gross revenue, net profit margins, transaction volumes, and cyclical day-of-the-week sales patterns.
* **Product & Attribute Analysis:** Automatic detection of top-performing items vs. slow-moving stock (Funnel Analysis) and multi-level product size distribution matrix.
* **Predictive Analytics & Forecasting (Tab 4):** Implementation of formal **Simple Linear Regression** built natively with NumPy to project future 7-day sales trends, combined with interactive **What-If Simulation Sliders** for operational target settings.

## 🛠️ Technology Stack
* **Core Language:** Python
* **Web Framework:** Streamlit
* **Data Manipulation:** Pandas & NumPy (Matrix computation for regression analytics)
* **Interactive Data Visualization:** Plotly Express & Plotly Graph Objects
* **Database Source:** Microsoft Excel (Multi-sheet processing)
